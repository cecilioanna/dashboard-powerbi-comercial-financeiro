# 📊 Dashboard Power BI - Análise Comercial e Financeira

## 🎯 Objetivo
Desenvolver um dashboard interativo para análise de desempenho comercial e financeiro, auxiliando na tomada de decisão estratégica.

## 📌 Estrutura do Projeto
O dashboard foi dividido em três visões:

- 📈 Comercial: análise de receita por produto, categoria e região  
- 💰 Financeiro: acompanhamento de custos, lucro e margem  
- 🧠 Executivo: visão estratégica com KPIs e insights  

## 📊 Indicadores principais
- Receita Total  
- Lucro  
- Margem (%)  
- Crescimento (%)  

## 🛠️ Ferramentas
- Power BI  
- DAX
